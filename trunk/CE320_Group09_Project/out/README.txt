README - INSTRUCTIONS
-----------------------------------------------------
The program is a picture collage generator. It will take pictures loaded by the user as an input and output a series of collages which the user can modify and then save. The user can also delete any collage they dislike

In the "src" folder in the "group9" package, run the class named "Piccollage" to start the application.

If a prompt is given to install any JAR files to run the program, use the JAR files stored in the "JARs" package in the "group9" package in the "src" folder
